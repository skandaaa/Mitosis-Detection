# Mitosis-Detection
An Explainable AI Decision Support System for Mitotic Detection using B-cos Networks
# Architecture
Frontend (Next.js + React)
Image Upload Component - Handles file uploads with preview
Results Component - Displays predictions and visualizations
Clean UI - Responsive design with Tailwind CSS
# Backend (FastAPI + PyTorch)
Model Loading - Flexible model integration
Inference Engine - Efficient prediction pipeline
Explainability - Grad-CAM heatmap generation
API Server - RESTful endpoints with CORS support
